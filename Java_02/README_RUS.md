# Project 02 — Java_Bootcamp

### Проект: Pet Info
В этом проекте рассматриваются ООП/процедурные/мультипарадигмальные подходы в Java, написание кода в соответствии с функциональной парадигмой, а также асинхронное/параллельное программирование при помощи проекта, который представляет собой набор модулей для отображения информации о питомцах.

## Задание 1. Список питомцев
Разработан модуль, который составляет список питомцев и выводит информацию о каждом питомце.

- Создан абстрактный класс Animal с двумя private-полями: строка имени питомца, целочисленный возраст.
- Реализован конструктор для абстрактного класса Animal, который принимает два параметра: строка имени питомца, целочисленный возраст и присваивает соответствующим полям передаваемые значения.
- Реализованы геттеры для private-полей: строка имени питомца, целочисленный возраст.
- Создан класс Dog, который наследуется от абстрактного класса Animal.
- Реализован конструктор для класса Dog, который принимает два параметра: строка имени питомца, целочисленный возраст и передает их в конструктор базового класса.
- Переопределен метод `toString()` в классе Dog, чтобы формировалась следующая строка: «Dog name = [имя_питомца], age = [возраст_питомца]».
- Создан класс Cat, который наследуется от абстрактного класса Animal.
- Реализован конструктор для класса Cat, который принимает два параметра: строка имени питомца, целочисленный возраст и передает их в конструктор базового класса.
- Переопределен метод `toString()` в классе Cat, чтобы формировалась следующая строка: «Cat name = [имя_питомца], age = [возраст_питомца]».
- Программа считывает количество питомцев.
- Программа считывает тип вводимого питомца: dog/cat.
- Каждый питомец добавляется в общий список pets.
- Если ввели неправильный тип питомца, то программа выводит: «Incorrect input. Unsupported pet type» и переходит к следующему вводу.
- Если ввели отрицательный или нулевой возраст, то программа выводит: «Incorrect input. Age <= 0» и переходит к следующему вводу.
